# 🧪 Arduino 기반 금속 반응 실험기 (초기 프로토타입)

## 📌 프로젝트 개요
본 프로젝트는 최소 부품(저항 없음, 비극성 콘덴서 없음) 환경에서  
코일의 전기적 특성 변화를 이용하여 금속 접근 시 반응을 확인하는  
초기 단계 금속 반응 실험 장치를 제작하는 것을 목표로 한다.

---

## 🎯 목표 정의

### 1차 목표 (필수)
- 금속(동전, 열쇠 등)이 코일 근처에 접근 시
- 측정값이 변화하는 것을 확인
- OLED 디스플레이에 수치 변화 표시

### 2차 목표 (선택)
- 기준값(Baseline) 자동 설정
- 변화량(Delta) 계산
- 임계값 초과 시 "METAL DETECTED" 표시

---

## 🔧 사용 부품

- Arduino Nano
- 브레드보드
- 점퍼선
- 구리 코일
- 전해 콘덴서
- OLED 디스플레이 (I2C)

---

## ⚠️ 제약 조건

- 저항 없음 → 전류 제한 및 안정성 낮음
- 비극성 콘덴서 없음 → 공진 특성 불완전
- 아날로그 회로 안정성 낮음
- 주변 금속/노이즈 영향 큼

👉 정밀 탐지기 X / 반응 실험기 O

---

## ⚙️ 시스템 구조

[코일 + 콘덴서] → [Arduino 입력핀] → [데이터 처리] → [OLED 출력]

---

## 🧠 동작 원리

1. 코일은 자기장을 생성
2. 금속이 가까워지면 자기장 변화 발생
3. 코일의 전기적 특성 변화
4. Arduino가 이를 측정
5. OLED로 출력

---

## 🪜 업무 진행 순서

### STEP 1: OLED 동작 확인
- I2C 연결
- 테스트 코드 업로드
- 출력 확인

---

### STEP 2: 코일 연결
- 코일 한쪽 → GND
- 다른쪽 → A0

---

### STEP 3: 콘덴서 추가
- 코일 주변 연결
- 극성 주의

---

### STEP 4: 데이터 읽기
- analogRead(A0)
- OLED 출력

---

### STEP 5: 기준값 설정
- 초기 평균값 저장

---

### STEP 6: 변화량 계산
DIFF = 현재값 - BASE

---

### STEP 7: 탐지 판정
if (abs(DIFF) > threshold)

---

### STEP 8: 안정화
- 이동 평균 적용

---
