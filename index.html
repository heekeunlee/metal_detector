OLED 모니터 연결하기

화면을 통해 센서 값을 실시간으로 확인해야 합니다. I2C 핀을 정확히 연결하세요.

- VCC → 5V
- GND → GND
- SCL → A5
- SDA → A4
2

코일과 콘덴서 회로

가장 중요한 센서 부분입니다. A0 핀에 병렬로 연결합니다.

- 콘덴서 (+) → A0
- 콘덴서 (-) → GND
- 코일 (무극성) → A0, GND 사이 연결
3

코드 업로드 (Coding)

아래 코드를 아두이노 IDE에 복사하여 업로드하세요. 방전 시간을 초단위로 측정합니다.

long pulse() {
  pinMode(A0, OUTPUT);
  digitalWrite(A0, HIGH);
  delayMicroseconds(10);
  pinMode(A0, INPUT);
  long t = micros();
  while(analogRead(A0) > 10);
  return micros() - t;
}

💡 과학 원리 (Theory)

와전류(Eddy Current): 금속이 변하는 자기장 속에 놓이면 금속 내부에 소용돌이 형태의 전류가 발생합니다. 이 전류는 에너지를 소비하며, 우리 센서의 '방전 시간'을 변화시킵니다.
